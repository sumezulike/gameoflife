Conway's Game of Life in JavaScript

Sorry, this applications needs JavaScript to work.



This is an implementation of Conway's Game of Life, a cellular automaton devised by mathematician John Conway in 1970. It uses the super-fast Hashlife algorithm, written in JavaScript using the canvas tag.

Forked from github.com/copy/life

Rules of Conway's Game of Life

The universe is an infinite two-dimensional grid of square cells, each of which is either alive or dead. Every cell interacts with its eight neighbors (horizontal, vertical, diagonal). At each step in time, the following transitions occur:

  1. Birth: A dead cell with exactly three live neighbors becomes alive
  2. Survival: A live cell with two or three live neighbors stays alive
  3. Death: A live cell with fewer than two neighbors dies (underpopulation)
  4. Death: A live cell with more than three neighbors dies (overpopulation)

These simple rules create surprisingly complex patterns and behaviors, including still lifes, oscillators, spaceships, and even patterns that can simulate universal computation.
